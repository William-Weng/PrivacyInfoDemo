新增隱私權文件

  1. 快速生成工具 - kimbely0320

  2. 新增文件

  3. 新增後會得到一個叫“PrivacyInfo.xcprivacy”的文件

隱私權文件設定

  1. Privacy Tracking Enabled - NSPrivacyTracking

    • 就是你到底有沒有去追踨人家

    • 如果有, 請選YES / 沒有，請選NO

  2. Privacy Tracking Domains - NSPrivacyTrackingDomains

    • 追踨的資料是送到哪裡？

    • 比如說是：NoPrivacy.com

  3. Privacy Nutrition Label Types - NSPrivacyCollectedDataTypes

    • 到底收集了什麼資料呢？

    • 包含第三方套件

  4. Privacy Accessed API Types - NSPrivacyAccessedAPITypes

    • 使用了什麼有關個人資料的API

    • 其中包含大家關心的：NSUserDefault

    1. Privacy Accessed API Type - NSPrivacyAccessedAPIType

      • 到底是用了哪一個API呢？

      • 比如說：最常用NSUserDefault

    2. NSPrivacyAccessedAPITypeReasons - Privacy Accessed API Reasons

      • 使用這個API的影響範圍

      • 比如說：測量用 / 存取資訊用 / 鍵盤資料用
